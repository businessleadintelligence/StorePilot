# Fact Generation

Facts are **deterministic** — no AI, no LLM calls. Generated by `app/knowledge/fact-builder/product-facts.ts`.

## Supported facts

| Fact | Trigger |
|------|---------|
| `InventoryLow` | qty ≤ 10 |
| `InventoryCritical` | qty ≤ 3 |
| `HighInventory` | qty ≥ 500 |
| `OutOfStock` | qty ≤ 0 |
| `NeverSold` | no matching order line items |
| `RecentlyPublished` | published ≤ 14 days |
| `MissingSEO` | no SEO title or description |
| `MissingMetaDescription` | no SEO description |
| `NoDescription` | empty body HTML |
| `MissingAltText` | media missing alt |
| `LowMediaCoverage` | fewer than 1 media item |
| `LowVariantCoverage` | fewer than 2 variants |
| `InactiveProduct` | non-active status |
| `DraftTooLong` | draft ≥ 30 days |
| `Discontinued` | archived status |
| `SingleProductCollection` | collection productCount ≤ 1 |
| `OrphanCollection` | collection productCount = 0 |
| `BundleCandidateSeed` | ≥ 2 variants |
| `SeasonalCandidate` | tag matches `/season/i` |
| `PriceChanged` | compareAtPrice ≠ price |
| `MarginRiskCandidate` | cost/price ≥ 0.7 |
| `PriceAboveCategoryAverage` | price > 125% of category avg |
| `RefundRiskSeed` | order totalRefundedAmount > 0 |

## Context inputs

- `soldVariantIds` — from order line items
- `categoryAveragePrice` — computed across catalog batch
- `observedAt` — pipeline timestamp

## Extension (Sprint 3+)

Add new facts by extending `buildProductFacts` / `buildOrderFacts` and updating readiness domain maps.
