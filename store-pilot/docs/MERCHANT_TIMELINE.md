# Merchant Timeline

- Recommendations and decisions
- Experiments and outcomes
- Predictions and validations
- Root cause confirmations
- Confidence changes
- Business DNA versions
- Business Memory versions
- Adaptive score updates

Future Business Simulation (Sprint 10+) consumes this timeline.
