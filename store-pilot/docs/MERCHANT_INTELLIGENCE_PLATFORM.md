# Merchant Intelligence Platform

StorePilot Sprint 9 — Adaptive Intelligence.

## Mission

Transform StorePilot from operational intelligence into a continuously improving business operating system. **No GPT. No new recommendations.** Only learns from merchant interactions and outcomes.

## Pipeline

```
Executive COO → Merchant Decision → Merchant Intelligence Platform → Business Memory → Business DNA → Future COO
```

Job: `merchant_intelligence_refresh` — scheduled after `executive_coo_generate` and on merchant experiment actions.

## Core principle

Merchant Intelligence **never creates intelligence** — it improves existing intelligence deterministically.

## Proprietary score

**Adaptive Intelligence (0–100)** — competitive differentiator based on merchant participation, journal coverage, experiment completion, prediction accuracy, memory coverage, and DNA maturity.

## Learning Attribution Engine

Every update carries provenance:

```
Business Outcome → Decision Journal → Evidence → Graph Nodes → Merchant Action → Learning Update → Memory Version → Future Recommendation
```
